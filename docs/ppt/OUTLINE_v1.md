## 🎬 制作工艺流程(整体工作流)

整体交付物有两个:**PPT 源文件** + **8 分钟答辩视频**。

```
  ┌──────────────────────────────────────────┐
  │ 1. 制作 PPT(基于本大纲,WPS/PowerPoint)   │
  │    - 10 页 + 完整内容(评委提问要展示细节)  │
  │    - 第 6、7 页先放 AI 配图占位            │
  └──────────────┬───────────────────────────┘
                 ↓
       ┌─────────────────────┐
       │ 2. PPT 转视频骨架     │  ← PowerPoint"另存为 MP4"
       │    - 每页 8-100 秒    │     或 OBS 录 PPT 放映
       │    - 配 AI 朗读       │
       └────────────┬─────────┘
                 ↓
       ┌─────────────────────┐
       │ 3. 剪映替换第 6、7 页 │  ← 用 demo1/demo2 final.mp4
       │    - 把录屏视频拖入   │     替换 AI 配图占位
       └────────────┬─────────┘
                 ↓
       ┌─────────────────────┐
       │ 4. 加字幕(剪映)      │  ← 智能字幕 → 校对
       └────────────┬─────────┘
                 ↓
       ┌─────────────────────┐
       │ 5. 加朗读配音         │  ← 剪映"文本朗读" / Edge TTS
       └────────────┬─────────┘
                 ↓
       ┌─────────────────────┐
       │ 6. 导出 8 分钟视频    │  ← 1920×1080 H.264 AAC
       └────────────┬─────────┘
                 ↓
            ☑ 视频完成

   ☑ PPT 源文件单独保存(用于评委提问、讨论、二次传播)
```

### ⏱️ 预估工时

| 阶段 | 预估 | 并行/串行 |
|---|---|---|
| 1. 制作 PPT | 2-3 小时 | 单人 |
| 2. PPT 转视频骨架 | 30 分钟 | 单人 |
| 3. 录 demo1 + demo2(录 + 剪辑) | 1-2 小时 | 单人 |
| 4. 剪映替换 + 字幕 + 配音 | 1-2 小时 | 单人 |
| 5. 校对 + 导出 | 30 分钟 | 单人 |
| **合计** | **5-8 小时** | 可 1-2 天做完 |

### 🛠️ 推荐工具链(国产优先)

| 工序 | 工具 | 说明 |
|---|---|---|
| 1. PPT | **WPS** | 免费、模板多、国产 |
| 2. PPT → MP4 | PowerPoint 自带"另存为" / OBS 录屏 | OBS 录屏可带鼠标高亮 |
| 3. 视频编辑 | **剪映**(Windows/Mac) | 国产免费,智能字幕、TTS 朗读 |
| 4. 录屏 | **OBS Studio** | 免费开源,1080p 30fps |
| 5. 配音 | **剪映"文本朗读"** / Edge TTS | 一个声音或多种,Edge TTS 免费 |
| 6. 配图生成 | **通义万相**(已配脚本) | `scripts/gen-ppt-images.py` |
| 7. 配图截帧 | **ffmpeg** | 录完视频后跑(见 images/README.md) |
