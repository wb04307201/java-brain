# PPT 配图 Prompt 清单(手工生成版)

> 用途:把每段 prompt 复制到任意 AI 配图工具(通义万相 / 即梦 / 智谱清影 / Midjourney)生成图片
> 命名:每张图必须按 `page-XX-主题.png` 命名,保存到 `images/` 目录
> 总数:8 张 AI 生成 + 2 张 ffmpeg 截帧(录完视频后)

---

## 使用方法

1. 选一个 AI 配图工具(下方有推荐)
2. 复制对应页的 prompt(中英两个版本任选,中文对国产工具更友好)
3. 生成图,下载到本地
4. 重命名 `page-XX-主题.png`
5. 放到 `images/` 目录

**推荐工具**:
| 工具 | 链接 | 特点 |
|---|---|---|
| **通义万相 Web** | https://tongyi.aliyun.com/wanxiang | 阿里出品,和你的 API 同源,免登录(扫码) |
| **即梦** | https://jimeng.jianying.com | 字节出品,中文理解强,免费额度多 |
| **智谱清影** | https://chatglm.cn | 智谱出品,CogView 模型 |
| **Midjourney** | midjourney.com | 风格最强,但要付费 |

## 兜底:为什么 API 跑不通(给用户存档)

`scripts/gen-ppt-images.py` 实测两个端点都失败:
1. `POST /api/v1/services/aigc/text2image/image-synthesis` → HTTP 403 `AccessDenied: current user api does not support synchronous calls`
2. `POST /compatible-mode/v1/images/generations` → HTTP 404 端点不存在

可能原因:
- dashscope 改版了 API,旧端点不再开放
- OpenAI 兼容模式只支持 LLM/embedding,不开放图像
- API key 没开通图像生成权限(loom-agent 只用了 chat + embedding)

后续可重新试:用户去阿里云百炼控制台查最新端点 + 开通图像生成权限,再改脚本重跑。
