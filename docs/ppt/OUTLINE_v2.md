## 全局设计

### 风格定位:**科技 + 视觉冲击**(深色 HUD 重音色)

整体调性不是"克制极简",而是"**让人一眼记住**":
- **数字要震**(关键数字 96-120pt,标题 32pt,正文 14-16pt —— 4-5 倍字号梯度)
- **色彩要撞**(红 vs 绿 / Java 蓝 vs AI 紫直接对位,不是"配合",是"对抗")
- **金句要空**(杀手锏金句独占 1/3 页面,周围大片留黑)
- **卡片要分层**(★ 杀手锏 2.5pt 金边 + 阴影,普通卡片 1pt 灰边,形成"明星 vs 龙套")
- **方向感要强**(关键元素加箭头 / 倾斜 / 光柱,避免全部圆角矩形方阵)

### 配色(深色 HUD,统一一套)
- 主色:Java 蓝 `#00D9FF`(灵梭) + AI 紫 `#A78BFA`(SQL 工坊)
- 撞色:钩子青绿 `#00FF9C`(杀手锏 / 成功) / 警示红 `#FF4D6D`(痛点 / 失败) / 警示金 `#FACC15`(路线图当前 / 重点)
- 背景:**深空黑 `#0A0E1A`** + 网格线 `#1A2238` + 卡片描边 `#1A2238`
- 文字:冷白 `#E4E7F1` + 冷灰蓝 `#7C8AA8` + 注脚 `#4A556E`

### 字号梯度(科技感的关键:数字 vs 文字 4-5 倍差)
- 杀手锏数字(痛点/反衬):**96-120pt mono**(占页 1/4)
- 核心金句:**40-48pt**(独占 1/3 页,周围留黑)
- 页面标题:**32-36pt**(顶部)
- 卡片标题:**18-22pt**
- 卡片副文 / 步骤 / 列表:**14-16pt**
- 注脚 / 路径:**10-11pt**

### 字体
- 中文:阿里巴巴普惠体 3.0
- 英文/数字:JetBrains Mono(代码/数字,杀手锏必须) + Inter(正文)

### 字体
- 中文:阿里巴巴普惠体 3.0
- 英文/数字:JetBrains Mono(代码/数字) + Inter(正文)
