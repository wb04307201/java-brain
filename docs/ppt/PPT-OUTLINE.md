## 🎬 制作工艺流程(整体工作流)

整体交付物有两个:**PPT 源文件** + **8 分钟答辩视频**。

```

  │    - 10 页 + 完整内容(评委提问要展示细节)  │
  │    - 第 6、7 页先放 AI 配图占位            │
  └──────────────┬───────────────────────────┘
                 ↓
       ┌─────────────────────┐
       │ 2. PPT 转视频骨架     │  ← PowerPoint"另存为 MP4"

       │    - 配 AI 朗读       │
       └────────────┬─────────┘
                 ↓
       ┌─────────────────────┐
       │ 3. 剪映替换第 6、7 页 │  ← 用 demo1/demo2 final.mp4
       │    - 把录屏视频拖入   │     替换 AI 配图占位
       └────────────┬─────────┘
                 ↓
       ┌─────────────────────┐
       │ 4. 加字幕(剪映)      │  ← 智能字幕 → 校对
       └────────────┬─────────┘
                 ↓
       ┌─────────────────────┐
```

### ⏯️ 导出设置(让每页时长不同)

PowerPoint / WPS 默认"另存为 MP4"时,**每页时长相同**。要让每页时长按本大纲分配,必须给每张幻灯片单独设置"自动换片时间":

| 软件 | 操作 |
|---|---|
| **PowerPoint** | `切换` 选项卡 → `计时` 组 → 勾选 `设置自动换片时间` → 填秒数 → `应用到全部`(**不要勾**——逐页设置) |
| **WPS 演示** | `幻灯片放映` → `排练计时` → 手动播放一次,系统记下每页时长,下次自动按此次数播放 |
| **录屏备选** | 用 OBS 录 PPT 放映 + 旁白(更灵活,可中途加动画停留) |

**关键原则**:
- **不要勾"应用到全部"**——每页时长按本页动画时序表单独设置
- 录屏页(6/7)用全屏视频,不要用 PowerPoint 动画撑时长,改用剪映合并
- 转场 2-5 秒通常不计入本页时长(由下一页开头的淡入抵消),但要在 PowerPoint `切换 → 持续时间` 里单独配

## 全局设计

### 🎨 整体风格

| 维度 | 设定 |
|---|---|
| 调性 | 科技感 + 商务感(企业级 AI 落地,**不卖萌不卡通**) |
| 视觉语言 | 2.5D 扁平 + 渐变色块 + 微立体阴影;**避免写实人物** |
| 节奏 | 静态为主(80% 时间),关键数据用动效高亮(20% 时间) |
| 信息密度 | 每页 **7±2 个元素**(米勒定律),留白要够,不要堆砌 |
| 字号层级 | 主标题 48pt / 副标题 24pt / 正文 16pt / 注脚 12pt(4 级严格递进) |
| 背景方案 | **统一用亮色版**(白底 #FFFFFF / 浅灰 #F8FAFC),深色只在录屏页 + 终止页用 |
| Logo 位置 | 固定右上角(尺寸 24×24pt),所有页一致 |

### 🖼️ 图片风格(给 AI 配图工具的"调性卡")

- **风格关键词**:`2.5D 扁平`、`几何块面`、`蓝色光晕`、`等距网格`、`微立体阴影`
- **主色调**:Java 蓝 `#007396` + AI 紫 `#6B46C1` + 白底;**禁止**暖色调(避免 AI 误生成橙红)
- **禁用**:写实人物、卡通形象、复杂背景、文字水印、3D 拟物、旋转动画
- **尺寸**:16:9 全屏(1920×1080);线稿图标 512×512 png 透明底
- **一致性**:同一组卡片图标保持**线条粗细 / 圆角弧度 / 投影深度**统一
- **负空间**:AI 图四周留 10% 空白,避免与文字争视觉
- **生成后处理**:用 ffmpeg 统一转 1920×1080 + sRGB + PNG 压缩(避免色差)

### 📐 页面布局(通用栅格)

- **栅格**:12 列,左右各 1 列安全边距(80px),上下各 80px
- **顶部固定栏**:`[Logo] — [节标题:第 N 页 · 主题] — [进度条 1/N]`,深灰 12pt
- **底部固定栏**:`[团队名 / GitHub] — [页码]`,浅灰 10pt
- **主区结构**(3 选 1,不要混用):
  - **左文 + 右图**(2:3):适合灵梭 / SQL工坊
  - **上文 + 下图**(5:4):适合仓库地图 / 路线图
  - **左栏 + 右栏**(1:1):适合实战 + 对比
- **录屏页特殊布局**:**全屏视频 + 顶部 1 行小字标注**(16pt 浅灰,40% 透明黑底)
- **对齐**:所有标题、卡片**左对齐 + 顶部对齐**,数字居中;禁用居中对齐的整页(显得业余)

### 🎞️ 动画风格(全 PPT 统一)

- **速度**:**0.4-0.6s 完成入场**;慢 = 拖沓,快 = 眼花
- **缓动**:`ease-out`(入场,减速到位)+ `ease-in`(出场,加速离开)
- **入场效果分级**(每页用 1-2 种,不要堆):
  - 主标题 / 数字:**缩放+淡入(0.6s)**,从 95% 缩到 100%
  - 卡片 / 图标:**缩放+淡入(0.5s)**,依次错开 0.3s 入场
  - 文字 / 说明:**纯淡入(0.4s)**
  - 杀手锏高亮:**金色脉冲**(边框 1.5px → 3px → 1.5px,2s 循环,**仅出现在讲解时**)
  - 数字滚动(从 0 滚动到目标值):**仅在痛点页用一次**(1.5s,适合"3 个月 vs 3 分钟")
  - 路径动画:**仓库连线用 dasharray 描边动画**(1s)
- **禁用**:旋转、3D 翻转、过场音效、粒子效果、闪烁(评委反感)
- **触发方式**:**全部"与上一动画同时"或"之后"**,不用"点击触发"(视频里没法点击)
- **自动换片**:每页用 `设置自动换片时间`,时长等于本页"🎬 旁白最末段时间"(不是总时长)
